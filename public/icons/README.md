# Icônes PWA

Placez ici les icônes suivantes pour la PWA :

- `icon-192x192.png` - Icône 192x192 pixels
- `icon-512x512.png` - Icône 512x512 pixels

Ces icônes doivent représenter le logo JobCamer avec les couleurs de la marque (vert émeraude #10B981).

Pour générer ces icônes, vous pouvez utiliser :
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

Ou créer une icône simple avec le texte "JC" sur fond vert dégradé.
